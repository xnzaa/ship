代码和流程都有问题
你先在VS2008的工具箱里拖拽一个SERIALPORT控件到窗口上，之后你点这个控件，看他的事件，事件其中有一个就是DataReceived，双击那个事件，自动就回跳转到事件的代码编辑界面，之后再写    watcher += sp.ReadExisting();MessageBox.Show(watcher);，之前需要把watcher这个公共变量声明一下,写 string watcher;       你需要屡清楚 代码的工作流程，从你的代码里感觉 你只知道要实现接收这个功能，但是接收这个流程你是不了解的，另外C#  有哪些控件，控件有哪些核心事件，这些都是你需要了解需要去学的。
